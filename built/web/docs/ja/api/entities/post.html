Post

投稿。

プロパティ

名前型オプション説明
created_atstring (Date)いいえ投稿日時
geoobject (geo)はい位置情報
idstring (ID)いいえ投稿ID
mediaobject[] (DriveFile)はい添付されているメディア
media_idsstring[] (DriveFile ID)はい添付されているメディアのID
my_reactionstringはいこの投稿に対する自分のリアクション
pollobject (poll)はい投票
reaction_countsobjectいいえリアクションをキーとし、この投稿に対するそのリアクションの数を値としたオブジェクト
replyobject (Post)はい返信した投稿
reply_idstring (Post ID)はい返信した投稿のID
repostobject (Post)はい引用した投稿
repost_idstring (Post ID)はい引用した投稿のID
textstringはい投稿の本文
userobject (User)はい投稿者
user_idstring (User ID)いいえ投稿者ID
via_mobilebooleanはいモバイル端末から投稿したか否か(自己申告であることに留意)

choice

名前型オプション説明
idnumberいいえ選択肢ID
is_votedbooleanはい自分がこの選択肢に投票したかどうか
textstringいいえ選択肢本文
votesnumberいいえこの選択肢に投票された数

geo

名前型オプション説明
accuracynumberいいえ緯度、経度の精度。メートル単位で表す。
altitudenumberいいえ高度。メートル単位で表す。
altitudeAccuracynumberいいえ高度の精度。メートル単位で表す。
headingnumberいいえ方角。0〜360の角度で表す。0が北、90が東、180が南、270が西。
latitudenumberいいえ緯度。-90〜90で表す。
longitudenumberいいえ経度。-180〜180で表す。
speednumberいいえ速度。メートル / 秒数で表す。

poll

名前型オプション説明
choicesobject[] (choice)いいえ投票の選択肢
